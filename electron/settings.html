Voice Activity Detection (VAD) Guide

What is VAD?

Voice Activity Detection (VAD) is a technology that automatically identifies when speech is present in audio recordings. It filters out silence, background noise, and non-speech sounds before sending audio to the transcription API.

Benefits

  • Save API costs: Only transcribe audio with actual speech, reducing unnecessary API calls
  • Improve performance: Faster processing by skipping empty or silent audio chunks
  • Better accuracy: Focus on actual speech content, reducing transcription errors
  • Reduce bandwidth: Less data sent to the API when silence is detected

Configuration Settings

Enable VAD: Toggle to turn VAD on or off. When disabled, all audio chunks will be transcribed regardless of content.

Energy Threshold: Controls how sensitive the system is to detecting voice.

  • Lower values (0.001-0.01): More sensitive - detects quieter speech but may include more background noise
  • Higher values (0.05-0.1): Less sensitive - only detects louder speech, filters out more noise
  • Recommended: Start with 0.02 and adjust based on your environment

Min Speech Duration: Minimum length of detected speech in milliseconds before processing.

  • Lower values (100-200ms): Detects very short speech fragments
  • Higher values (500-1000ms): Only processes longer speech segments
  • Recommended: 300ms is a good balance for most use cases

Silence Threshold: Threshold below which audio is considered silence.

  • Lower values (0.001-0.002): Very strict silence detection
  • Higher values (0.005-0.01): More lenient, allows quieter sounds
  • Recommended: 0.003 works well for most environments

How to Use

  1. Enable VAD: Check the "Enable VAD" checkbox (recommended)
  2. Test your environment: Start with default values and record some audio
  3. Adjust if needed: If VAD is too sensitive (detects background noise), increase Energy Threshold
  4. If missing speech: If VAD misses some speech, decrease Energy Threshold
  5. Save and restart: After changing settings, save and restart the app for changes to take effect

Important Notes

  • Settings changes require an app restart to take effect
  • VAD works best in quiet environments with clear speech
  • Very noisy environments may require higher thresholds
  • If VAD is disabled, all audio will be transcribed (higher API costs)
  • Test with different values to find what works best for your setup
