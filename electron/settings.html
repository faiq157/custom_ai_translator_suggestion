Configure your Meeting AI Assistant

API Key Required: You need an OpenAI API key to use this application. Get one at platform.openai.com
Audio Permissions: The app needs microphone access to transcribe your meetings.
Audio Capture Modes:
• Microphone Only: Your voice only (self-contained)
• System Audio: Desktop apps (Slack, Teams, Zoom) + your mic
System Audio Requirements:
Linux: PulseAudio (pre-installed) ✅
Windows: Enable "Stereo Mix" in Sound settings
macOS: Install BlackHole (free virtual audio device)
Works with: Slack desktop, Teams desktop, Zoom desktop, etc.

Voice Activity Detection (VAD)

VAD filters out silence and background noise before calling Whisper API, saving costs and improving performance.

Local Server: The app runs a local server on your computer. No data is sent to external servers except OpenAI API calls.
